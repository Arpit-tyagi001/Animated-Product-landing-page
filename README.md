# Animated Ray-Ban Landing Page

An animated Ray-Ban inspired product landing page built using HTML, CSS, JavaScript, GSAP, and Shery.js. This project features smooth scroll-based animations, interactive image effects, and modern UI transitions to create an engaging product showcase experience.

The project was built as a learning exercise by following and understanding concepts taught on the Sheryians Coding School YouTube channel while implementing the animations and interactions independently.

## Features

* Smooth GSAP animations
* Shery.js interactive effects
* Animated product showcase
* Modern landing page design
* Clean and responsive layout
* Desktop/Laptop optimized experience

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP
* Shery.js

## Note

This project is currently optimized for desktop and laptop screens. Mobile responsiveness is not fully implemented yet.
