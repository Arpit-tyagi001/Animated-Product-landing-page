# 🕶️ Animated Ray-Ban Landing Page

A modern, animation-rich **Ray-Ban inspired product landing page** built with **HTML, CSS, JavaScript, GSAP, and Shery.js**. The project focuses on creating a visually engaging user experience through smooth animations, interactive effects, and elegant UI transitions.

This project was developed as part of my frontend development journey by learning animation concepts from the **Sheryians Coding School** YouTube channel and implementing them to strengthen my understanding of modern web animations.

---

## ✨ Features

* 🎯 Smooth GSAP-powered animations
* 🎨 Interactive visual effects using Shery.js
* 🕶️ Animated product showcase
* ⚡ Modern and minimal landing page UI
* 📜 Smooth scrolling experience
* 💻 Clean, responsive desktop layout
* ✨ Engaging micro-interactions and transitions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP
* Shery.js
